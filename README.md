# 📊 HR Analytics Dashboard (Tableau)

## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive **Tableau data visualization project** designed to analyze employee workforce data and generate meaningful insights for HR decision-making.

This dashboard transforms raw HR data into a visual analytical tool that helps organizations monitor **employee distribution, attrition trends, demographic patterns, and departmental performance**.

The project demonstrates how **Tableau can be used to convert HR datasets into actionable business intelligence insights**.

---

# 🎯 Project Objective

The main objective of this project is to build a **dynamic HR dashboard in Tableau** that enables HR teams and business leaders to:

* Track employee distribution across departments
* Monitor attrition trends
* Analyze workforce demographics
* Identify patterns affecting employee retention
* Support data-driven HR decisions

---

# 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

* **Tableau** – Data visualization and dashboard development
* **Data Cleaning** – Preparing the HR dataset for analysis
* **Calculated Fields** – Creating custom metrics and KPIs
* **Filters & Parameters** – Interactive data exploration
* **Dashboard & Worksheet Design** – Creating clear and informative visualizations

---

# 📂 Data Source

The dataset contains **employee HR records** including key attributes such as:

* Employee ID
* Department
* Job Role
* Gender
* Age
* Education
* Attrition Status
* Salary
* Years at Company

This dataset allows multi-dimensional analysis of workforce trends and employee behavior.

---

# 📊 Key Dashboard Insights

### 👥 Employee Distribution

Shows how employees are distributed across **different departments and job roles**, helping HR understand workforce structure.

### 📉 Attrition Analysis

Highlights **employee attrition patterns**, allowing HR teams to identify departments or roles with higher turnover.

### 🎓 Workforce Demographics

Visualizes employee demographics including:

* Age groups
* Gender distribution
* Education levels

### 📈 Department-Level Insights

Analyzes workforce performance across departments to identify areas requiring attention.

### 📊 Attrition by Age Group

Provides insight into **which age groups are more likely to leave the organization**.
